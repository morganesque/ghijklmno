New Computer

— 22nd July 2004

Got a new computer from MESH and it’s delicious. A little little shuttle box with a big big Iiyama screen and lovely all-in-one style printer/scanner. Very much more neat and tidy than the last ’puter I had, and with Windows XP too which, although it’s Microsoft (and so obviously evil), I do find surprisingly easy to make work for me.

So all in all in all I was very happy. It also has an ethernet port (which the old one didn’t), so I can connect to my router via a network cabel (yay) and not through the USB interface I was using before (boo). Or so I thought...
