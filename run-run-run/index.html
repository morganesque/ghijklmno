Run Run Run

— 15th October 2002

OK time to out myself. Time to leap from my closet and tell the world. I have something to say, it may surprise you, it may disgust you, but I feel I’ve kept silent too long.

I’ve been jogging

No, I don’t mean I’ve done it in the past. I mean I’ve been doing it again. It began just over a week ago (the weekend before last). We’d been talking about getting some exercise (I’ve been talking about it for years now) and we’d even discussed investing in one of those swish treadmill running ma-thingies which we decided we could just about squeeze in. However before we went all out with the buying, it was decided that we should maybe try the normal, real, outside, proper sort of running instead. And so it was, that last Sunday morning we set off and ran round the block. It took 15 mins and it was actually quite good fun (this was probably mainly to do with the lovely crisp autumn weather and less to do with the jogging, but none the less...). So the next morning I did it again. And the next, and the next.

Now I know what you’re thinking. "Whoah there sonny, if you’re not careful you’ll end up fit or something". Well yes I am aware of that and rest assured I’ll be upping my ice cream and beer intake to compensate.
